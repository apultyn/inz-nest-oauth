# Nest.js API

## Komendy

- prisma studio: `npx prisma studio`
- generowanie modułu/serwisu: `nest g <service/module> <nazwa> --no-spec`
- `--no-spec` jak nie chcemy testów
- `npm run start:dev` (uruchamia się po cichu na localhost:8080)
- `npm run test:e2e` (trzeba mieć bazę na localhost 3307)
- `npx prisma migrate deploy`
